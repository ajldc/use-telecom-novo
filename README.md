# Use Telecom — Novo Site

Proposta de redesign do site institucional da Use Telecom.

## Stack
- HTML estático (single-file)
- Tipografia: Bricolage Grotesque + Inter Tight + JetBrains Mono (Google Fonts)
- Identidade visual: magenta Use Telecom (#E20074), inspirada no padrão T-Mobile
- Hospedagem: Cloudflare Pages
- Domínio de teste: novo.usetelecom.com.br

## Conteúdo
- Hero com posicionamento "Conectividade que sustenta o que não pode parar"
- 4 verticais: Use Casa, Use Condomínio, Use Empresa, Use Eventos
- Caso de destaque: Wi-Fi do Leão (Estádio Barradão)
- Diferenciais técnicos: backbone próprio, PTT, operação local
- Cobertura: 48 cidades + verificador de CEP
- Parceiros: EC Vitória, Globoplay, Nokia, Campus Party
- Footer institucional

## Deploy
Site estático puro. Não requer build. Basta apontar o root do projeto para o Cloudflare Pages.

## Status
Versão 1 — proposta visual. Aguardando validação para desenvolvimento das páginas internas.
